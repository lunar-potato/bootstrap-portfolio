Technical Skills

From the bootcamp, I want to learn more of the technical aspects of front-end development. Learning the languages needed for web development and getting to study HTML & CSS, JavaScript, Frameworks, APIs, TypeScript, REACT, etc. There are multiple things to learn under the technical skills for front-end development and with the bootcamp, I hope to learn what is needed for web development or front-end development. Without the technical skills it would be a difficult path to go for front-end development.

Creativity

In terms of this skill, it is important to have creativity for front-end development. With good creativity, this will deem a website usable and good to look at. Creativity is useful in CSS and JS which ensures that the website looks presentable and accessible for users. Being a developer means that the basis of creativity starts with solving problems, how a developer would do when problems arise and what solutions one would do to fix any problems or bugs. Which is why with the bootcamp, I hope to expect that my creativity skills will be boosted.

Teamwork

With front-end development, being able to communicate with others and working in a team is crucial since creating a full-blown wesbite would require working in a team to be able to finish the website on time and be able to make the website according to what is required and needed. Being a solo player would not be impossible but it would take too much time and having an extra pair of eyes would allow developers to see errors and bugs early on. I do hope that I get to be a better teammate for others and that I would not be afraid of communicating with others.

Problem-Solving

With development, problem-solving is needed to resolve any technical diffulties or technical issues. Without problem-solving it would be difficult to fix any errors or bugs that may appear on your code. With errors, it does not have a specific solution or answer to fix, with great mindset and problem solving, any errors can be fix with perseverance. And with this I hope that I can improve my problem-solving skills.
